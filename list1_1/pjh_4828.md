# SWEA 4828. [파이썬 S/W 문제해결 기본] 1일차 - min max

## 문제
N개의 양의 정수에서 가장 큰 수와 가장 작은 수의 차이를 출력하시오.

### 입력

첫 줄에 테스트 케이스의 수 T가 주어진다. ( 1 ≤ T ≤ 50 )

각 케이스의 첫 줄에 양수의 개수 N이 주어진다. ( 5 ≤ N ≤ 1000 )

다음 줄에 N개의 양수 ai가 주어진다. ( 1 ≤ ai≤ 1000000 )

### 출력

각 줄마다 "#T" (T는 테스트 케이스 번호)를 출력한 뒤, 답을 출력한다.

## 코드 방향성 / 구상한 내용
오늘 배운 버블 정렬을 활용하려고 노력함

## 어려웠던 점
딱히 없음;

```python
import sys

sys.stdin = open('4828.txt')

# 테스트 케이스 개수
T = int(input())

# 테스트 케이스 반복
for tc in range(1, T+1):
    N = int(input())
    ai = list(map(int, input().split()))
    
		# ai리스트의 0번째 인덱스를 각각 max와 min val로 지정
    max_val = ai[0]
    min_val = ai[0]
    for number in range(N):
		    # N을 순회하며 max_val, min_val 값과 비교
		    # ai[number]가 큰 경우 max_val
		    # ai[number]가 작은 경우 min_val
        if max_val < ai[number]:
            max_val = ai[number]
        if min_val > ai[number]:
            min_val = ai[number]
            
    print(f'#{tc} {max_val - min_val}')
```
