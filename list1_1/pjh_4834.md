# SWEA 4834. [파이썬 S/W 문제해결 기본] 1일차 - 숫자 카드

## 문제
0에서 9까지 숫자가 적힌 N장의 카드가 주어진다.

가장 많은 카드에 적힌 숫자와 카드가 몇 장인지 출력하는 프로그램을 만드시오. 카드 장수가 같을 때는 적힌 숫자가 큰 쪽을 출력한다.

### 입력

첫 줄에 테스트 케이스 개수 T가 주어진다.  ( 1 ≤ T ≤ 50 )

다음 줄부터 테스트케이스의 첫 줄에 카드 장수 N이 주어진다. ( 5 ≤ N ≤ 100 )

다음 줄에 N개의 숫자 ai가 여백없이 주어진다. (0으로 시작할 수도 있다.)  ( 0 ≤ ai ≤ 9 ) 

### 출력

각 줄마다 "#T" (T는 테스트 케이스 번호)를 출력한 뒤, 가장 많은 카드의 숫자와 장 수를 차례로 출력한다.

## 코드 방향성 / 구상한 내용
1. 오늘 배운 버블 정렬을 사용하려고 노력함
2. 가장 많은 카드와 그 카드가 몇 장인 지 출력해야 하므로 딕셔너리 사용하려고 생각함
3. 근데 카드 장수가 같을 때는 적힌 숫자가 큰 쪽을 출력해야 하므로 리스트를 활용해야 하긴함
4. 그래서 순서를 str -> int -> list -> dict 이렇게 구상
## 어려웠던 점
처음에 카드 장수가 같을 때 숫자 큰 쪽 << 이걸 못 봐서 좀 당황했었음.

```python
import sys

sys.stdin = open('4834.txt')

# 테스트 케이스 개수
T = int(input())

# 테스트 케이스 반복
for tc in range(1, T + 1):
    N = int(input())
    # 카드 숫자들이 문자열로 들어옴
    ai = input()

    # 카드 숫자를 정수로 저장할 리스트
    ai_list = []

    # 각 문자를 int로 바꿔 리스트에 저장
    for idx in ai:
        ai_list.append(int(idx))

    # 카드 숫자를 오름차순 정렬
    # (동점일 때 큰 숫자를 뽑기 위함)
    ai_list.sort()

    # {카드 숫자: 등장 횟수}
    num_dict = {}

    for num in ai_list:
        #처음 등장한 숫자라면 key 생성
        if num not in num_dict:
            num_dict[num] = 0
        # 등장 횟수 증가
        num_dict[num] += 1

    max_num = 0  # 가장 많이 등장한 횟수
    max_key = 0  # 그 횟수를 가진 카드 숫자

    for key, value in num_dict.items():
        # 등장 횟수가 더 크거나 같으면 갱신
        if value >= max_num:
            max_num = value
            max_key = key


    print(f'#{tc} {int(max_key)} {int(max_num)}')
```
